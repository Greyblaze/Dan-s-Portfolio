E-commerce Site

Description: An E-commerce website for sport products. The 'home' section contains a slide of 3 exclusive products on offer. The 'featured' section involves the main image of a particular product along with 4 smaller images showing various angles of the product nested within the same container - made with flex properties. A customer review section was also built. The menu-bar was created using Javascript to present the navigation bar, creating a better effect on smaller devices.
